# **App Name**: VOSEM International

## Core Features:

- Sermon Summaries: Provide AI-generated summaries of past sermons.
- Event Calendar: Display a calendar of upcoming events with detailed information.
- Testimonies: Showcase written testimonies from church members.
- Online Giving: Enable online donations via secure payment gateways such as Paystack and Flutterwave.
- Interactive Prayer Requests: Allow users to submit prayer requests which can be shared (anonymously) on a prayer wall.

## Style Guidelines:

- Primary color: Deep purple (#7c28c5) to evoke a sense of reverence and spirituality. (From the original Tailwind config.)
- Background color: Light gray (#f6f6f8) in light mode and dark charcoal (#0a0a1a) in dark mode. (From the original Tailwind config.)
- Accent color: Magenta (#b50452) for interactive elements and highlights. (From the original Tailwind config.)
- Body font: 'Plus Jakarta Sans', a sans-serif, for a clean and modern feel. (As specified by the user.) Note: currently only Google Fonts are supported.
- Headline font: 'Playfair Display', a serif, for elegant and readable headlines. (As specified by the user.) Note: currently only Google Fonts are supported.
- Use Material Symbols Outlined for clear and consistent icons. (As specified by the user.)
- Maintain a clean, modern layout with clear sections for easy navigation.
- Use subtle transitions and hover effects to enhance interactivity.
